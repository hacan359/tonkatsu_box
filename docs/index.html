Get Started

Up and running in minutes

Grab free API keys, install the app, and start building your collection.

01

Get API Keys

Register for free at IGDB (via Twitch) for games and TMDB for movies & TV. Optional: SteamGridDB for extra artwork.

02

Install & Run

Download the latest release or build from source with Flutter. Available for Windows desktop and Android.

03

Start Collecting

Enter your API keys in Settings, search for your favorite titles, and organize them into collections. That's it!

Sharing

Share your taste

Export your collections and send them to friends. Two formats for different needs.

.xcoll
Light Export

Small file with the list of items. Recipient needs internet to load images and details.

.xcollx
Full Export

Complete package with images, board layouts, and all media data. Works fully offline.
